## Getting Help

### Before Asking

1. **Check Documentation:**
   - [README.md](../../README.md)
   - [TEST_ENVIRONMENT_GUIDE.md](../../TEST_ENVIRONMENT_GUIDE.md)
   - [QUICKSTART.md](../../QUICKSTART.md)

2. **Search Existing Discussions:**
   - Use GitHub Discussions search
   - Check closed issues
   - Review FAQ sections

3. **Review Examples:**
   - [Example Scripts](../../blockchain/scripts/)
   - [Test Reports](../../test_outputs/)
   - [Paper Discovery Summary](../../test_outputs/PAPER_DISCOVERY_SUMMARY.md)

### How to Ask

When asking for help, include:
- What you're trying to do
- What you've tried
- Error messages (if any)
- System information (OS, Node version, etc.)
- Relevant code snippets
